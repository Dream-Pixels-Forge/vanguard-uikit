# Vanguard UIKit Demo

A comprehensive demo showcasing all components from the Vanguard UIKit.

## Quick Start

```bash
cd demo
npm install
npm run dev
```

## Features Demo

- **Buttons** - All variants and sizes
- **Cards** - Feature cards, stat cards, pricing, testimonials
- **Forms** - Inputs, selects, toggles, sliders, checkboxes
- **Data** - Tables, progress, accordions, tabs, charts
- **Feedback** - Modals, toasts, dropdowns, tooltips, loaders
